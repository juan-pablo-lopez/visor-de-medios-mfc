# Flashcards

Las flashcards se guardan por tema en un directorio:

```
flashcards/<topic>/
<card-id>.json
<card-id>.json
...
```

Para referencia, ver la carpeta `tema-7` como ejemplo.

**NOTA**: es recomendable crear un archivo `SVG` con el mismo nombre de la carpeta y que identifique el tema del conjunto de tarjetas en:

```
/frontend/public/flashcards
```
